# AI Powered Content Manager

## Developed By
**Abinaya S**

## Project Overview
AI Powered Content Manager is a web application that allows users to generate, manage, update, and delete AI-generated content. It is built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## Features
- AI Content Generation
- Create, Read, Update, Delete (CRUD)
- Responsive User Interface
- MongoDB Database Integration
- REST API Support

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose

## Installation
1. Clone the repository.
2. Install dependencies:
   ```
   npm install
   ```
3. Start MongoDB.
4. Run the server:
   ```
   node server/server.js
   ```
5. Open `client/index.html` using Live Server.
